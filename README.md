# EmbyChineseNameSynchronous
Emby 英文媒体标题自动同步中文标题和中文人名


*  注意使用本工具需要emby本身刮削了tmdb的完整数据，工具只是获取原有的数据进行替换
1. 配置文件config/config.yaml
2. 建议使用Python3
3. 安装依赖模块
*  pip install zhconv
*  pip install requests
*  pip install json
*  pip install yaml
*  pip install logging
