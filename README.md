# EmbyChineseNameSynchronous
Emby 英文媒体标题自动同步中文标题和中文人名


0. 注意使用本工具需要emby本身刮削了tmdb的完整数据，工具只是获取原有的数据进行替换
1. 使用说明看main.py文件，配置完成直接运行main.py即可
2. 建议使用Python3
3. 安装依赖模块
4. pip install zhconv
5. pip install requests
6. pip install json
